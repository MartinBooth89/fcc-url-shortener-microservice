A Url Shortener

Usage

/shorten/url

url: a valid URL to shorten, e.g. http://a.b.c

result: returns a JSON object in the format {"original_url": "http://a.b.c", "short_url": 1234}

error: if the url is not valid


/visit/short-url

short-url: an integer returned by /shorten/url, e.g. 1234

result: redirects to original url, e.g. http://a.b.c

error: if the short-url does not exist, or if the short-url is not an integer
